# 🌐 四级翻译常用句型

> 翻译目标：**意思写出来就得分，不需要高级词汇！**

---

## 🧩 翻译五步法

```
中文原句 → ①找主干(主谓宾) → ②切分句子 → ③先译主干 → ④加修饰 → ⑤检查
```

### 示例：
```
中文：位于北京市中心、有着600多年历史的故宫，每天都吸引着成千上万的游客。

① 找主干：故宫吸引游客
② 切分：位于市中心 / 有600年历史 / 吸引游客
③ 主干：The Forbidden City attracts tourists.
④ 加修饰：
   → Located in the center of Beijing,
   → With a history of over 600 years,
   → thousands of tourists every day.
⑤ 最终：Located in the center of Beijing, the Forbidden City,
   with a history of over 600 years, attracts thousands of tourists every day.
```

## 📐 十大必背句型

### 1. 「是...的」= It is... that...（强调句）
```
正是通过努力他才取得了成功。
→ It was through hard work that he achieved success.
```

### 2. 「越来越...」= more and more / increasingly
```
人们越来越关注健康问题。
→ People are paying more and more attention to health issues.
```

### 3. 「不仅...而且...」= not only... but also...
```
读书不仅增长知识，而且开阔视野。
→ Reading not only increases knowledge but also broadens horizons.
```

### 4. 「据说/据报道」= It is said/reported that...
```
据说这个传统已经延续了几千年。
→ It is said that this tradition has lasted for thousands of years.
```

### 5. 「...被用来...」= ... is used to...
```
筷子被用来吃饭已经有几千年了。
→ Chopsticks have been used for eating for thousands of years.
```

### 6. 「作为...」= As...
```
作为一个古老的文明古国，中国有着丰富的文化遗产。
→ As an ancient civilization, China has rich cultural heritage.
```

### 7. 「...的特点是...」= ... is characterized by...
```
中国饮食的特点是品种多样、味道丰富。
→ Chinese cuisine is characterized by its variety and rich flavors.
```

### 8. 「...起源于...」= ... originated from / dates back to
```
茶文化起源于中国，距今已有4000多年。
→ Tea culture originated from China, dating back over 4,000 years.
```

### 9. 「...被看作是...」= ... is regarded/viewed as...
```
长城被看作是最伟大的建筑之一。
→ The Great Wall is regarded as one of the greatest architectures.
```

### 10. 「在...中扮演重要角色」= plays an important role in...
```
互联网在我们的日常生活中扮演着重要角色。
→ The Internet plays an important role in our daily life.
```

## 🆘 不会的词怎么办？3招保命

| 方法 | 示例 |
|------|------|
| **上位词代替** | 包子 → food / 沙发 → furniture |
| **解释性表达** | 针灸 → a medical treatment using needles |
| **省略不译** | 如果那个词不影响主干意思，就大胆跳过！ |

```
原文：人们在元宵节吃汤圆，象征团圆。
你的翻译：People eat sweet dumplings on the Lantern Festival,
           which symbolizes family reunion.
（即使不知道"汤圆"怎么写，用 sweet dumplings 也完全得分！）
```

## ⏱️ 翻译时间分配

```
0-2min：  通读中文全文，画出主干
2-25min： 逐句翻译（不会的词先跳过）
25-28min：检查时态、单复数、拼写
28-30min：通读一遍，查漏补缺
```

## 🔍 常见错误自查清单

| 错误类型 | 示例 | 纠正 |
|----------|------|------|
| 时态混乱 | He go to school. | He **goes** to school. |
| 单复数 | many student | many **students** |
| 中式英语 | very like | **really** like |
| 缺主语 | Is important to study. | **It** is important to study. |
| 冠词遗漏 | I went to hospital. | I went to **the** hospital. |
| 流水句 | I like reading, it is fun. | I like reading **because** it is fun. |
