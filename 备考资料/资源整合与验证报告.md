### 🎧 听力补充技巧

#### ① 新闻题「前3句原则」
> 新闻听力90%的答案出现在前3句话中（导语）。所以每段新闻开始时要高度集中！

#### ② 选项预读清单（播音前30秒做的事）
```
1. 横读：扫一遍所有选项，知道大概话题
2. 竖读：比较4个选项的差异（哪个词不一样？）
3. 圈出：人名、数字、时间、否定词（not/never/hardly）
4. 预判：可能问什么？（原因？结果？态度？数字？）
```

#### ③ 听力常见「陷阱词」
| 听到的 | 选项里变成 | 含义 |
|--------|-----------|------|
| but / however | 转折后是重点 | 前面说的都不算 |
| actually / in fact | 这才是真话 | 前面的可能是否定 |
| I'm afraid | 委婉拒绝 | 不是害怕 |
| why not | 强烈建议 | 不是提问 |
| I'd love to, but... | 委婉拒绝 | 后面的but是关键 |

### 📖 阅读补充技巧

#### ① 仔细阅读「题文同序」
> 第1题的答案在第1-2段，第2题在第3-4段……题目顺序 = 文章段落顺序，绝不跳序！

#### ② 长篇匹配「2+1原则」
> 一段最多匹配2个句子，不可能一段匹配3题以上。如果某段已经勾了2题，其他题直接排除这段。

#### ③ 选词填空「词性排除法」
```
1. 先把15个选项标上词性：n./v./adj./adv.
2. 看空格前后判断需要的词性：
   - a/an/the + ___ + n. → 填adj.
   - be + ___ + by → 填v.-ed（被动）
   - to + ___ → 填v.原形
   - 句首缺主语 → 填n.
3. 在符合条件的2-3个词里选
```

### ✍️ 作文补充技巧

#### ① 刘晓燕极简模板（比我的还简单）

```
第一段（2句）：
Nowadays, [话题] has become a hot topic.
Different people have different opinions on this issue.

第二段（3-4句）：
Some people think that [观点A]. They believe that [理由].
However, others hold the opposite view. They argue that [理由].
As for me, I agree with the former/latter.

第三段（2句）：
In conclusion, [你的观点].
Only in this way can we [展望].
```

### 🆘 终极蒙题技巧

> 如果实在来不及/不会做，用这些统计规律：

1. **仔细阅读**：正确选项通常是原文的「同义改写」，照抄原文的往往是干扰项
2. **听力**：ABDC四个选项在25题中大致均匀分布（约6-7题每个），如果C选得特别少，后面蒙C
3. **选词填空**：全部选同一个词性（建议都选名词或都选动词），至少对2-3题
4. **长篇匹配**：如果某段找不到对应的句子，跳过，先做其他段
5. **绝对词排除**：选项中有must/never/all/none的往往是错的
