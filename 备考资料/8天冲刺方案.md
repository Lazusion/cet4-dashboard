## ⚡ 各模块应试技巧速成

### 🎧 听力（35% | 提分最快）

**关键认知：听不懂也能对一半！**

1. **预读选项** → 播音前30秒扫读选项，圈关键词
2. **首句必听** → 新闻/对话的开头前半句通常就是答案
3. **同义替换** → 听到的词往往不直接出现在选项中，而是换成同义词
4. **顺序原则** → 题目顺序 = 听力内容顺序，别跳题
5. **视听一致** → 听到最多的那个选项往往是正确答案

**每日任务：**
- 精听1套真题（先做→对答案→再听一遍看原文）
- 泛听30分钟（走路/吃饭时磨耳朵）

### 📖 阅读（35% | 做题顺序决定生死）

**⏱️ 黄金做题顺序：C → B → A**

| 顺序 | 题型 | 原因 |
|------|------|------|
| 1️⃣ | 仔细阅读C（10题/20%） | 分值最高，每题14.2分！ |
| 2️⃣ | 长篇匹配B（10题/10%） | 技巧性强，快速拿分 |
| 3️⃣ | 选词填空A（10题/5%） | 性价比最低，有时间再做 |

**仔细阅读（Section C）技巧：**
- 先读题干→定位段落→对照选项
- 正确答案通常是原文的"同义改写"
- 太绝对的选项（must/never/all）往往是错的

**长篇匹配（Section B）技巧：**
- 先读10个句子，圈关键词（人名/数字/大写/名词）
- 快速扫读文章段落，找关键词对应
- 一段可能匹配多题，一题只匹配一段

**选词填空（Section A）技巧：**
- 先标出15个选项的词性（n./v./adj./adv.）
- 根据空格前后语法判断需要什么词性
- 缩小范围后再选

### ✍️ 写作（15% | 模板是生命线）

**核心策略：背熟3套模板，考场上直接套用！**

三种常考类型：
1. 议论文（观点对比型）
2. 图表/现象分析型
3. 应用文（书信/通知）

→ **详见《作文万能模板》**

### 🌐 翻译（15% | 意思到了就行）

**翻译5步法：**
1. 找主干（主谓宾）
2. 切分句子（中文一句→英文可能两到三句）
3. 先翻主干，再加修饰
4. 不会的词：上位词/解释性表达/省略
5. 检查时态和单复数

→ **详见《翻译常用句型》**

## 🔬 补充技巧（刘晓燕+考虫+新东方汇总）

### 听力额外技巧
- 📻 **新闻前3句原则：** 90%的答案在每段新闻前3句
- 🔍 **预读30秒：** 横读→竖读→圈名词/数字/否定词→预判问题
- 🚫 **陷阱词：** but/however后面是重点 / actually才是真话 / I'm afraid=委婉拒绝

### 阅读额外技巧
- 📐 **题文同序：** 仔细阅读题目顺序 = 文章段落顺序
- 🔢 **2+1原则：** 一段最多匹配2题，不可能3题以上
- 🏷️ **词性排除法：** 选词填空先标15词词性，再看空格语法需求

### 写作额外技巧
- ✂️ **刘晓燕极简模板：** 词汇量不足也能用（见作文模板文件）
- 📈 **王江涛词汇升级：** but→however, so→therefore, many→numerous
- ⚠️ **逻辑词位置规则：**
  - 句首类（However/Therefore/Moreover…）→ 后面**必须加逗号**
  - 句中类（because/since/although/while…）→ **不加逗号**
  - 不确定就往句首放，句首+逗号永远正确！

### 蒙题统计规律
- 听力ABCD每项约6-7题，分布均匀
- 选词填空全蒙同一个词性至少对2-3题
- 选项含must/never/all/none的往往是错的
- 正确选项是原文的同义改写，照抄原文的往往是干扰项
