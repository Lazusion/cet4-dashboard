## ✍️ 写作文时怎么用

**策略：先全写简单句 → 用关系词串起来 → 替换升级词**

```
原始（全是简单句）:
I like reading. Reading is fun. I learn a lot. I read every day.

加关系词后:
I like reading because it is fun. Moreover, I learn a lot from it.
Therefore, I read every day.

替换升级词:
I like reading because it is fun. Furthermore, I acquire a great deal 
of knowledge from it. As a result, I read on a daily basis.
```
