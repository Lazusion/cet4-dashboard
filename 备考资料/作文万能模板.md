# ✍️ 四级作文万能模板

> 背熟这3套模板，考场上直接套！任选其一即可。

---

## 📐 模板一：观点对比型（最常考！）

**适用题目：** Should we...? / Some people think... others believe... / The advantages and disadvantages of...

```
[第一段：引出话题 + 提出争议]
Recently, the issue of [话题] has aroused wide concern among the public.
Some people hold the view that [观点A], while others argue that [观点B].
Personally, I tend to support the former/latter.

[第二段：论述己方观点（2个理由）]
There are several reasons accounting for my choice.
To begin with, [理由1]. For example, [举例].
Furthermore, [理由2]. It is widely acknowledged that [展开].

[第三段：承认对方 + 重申立场]
Admittedly, [对方观点的合理之处]. However, this does not mean that
[对方观点不如己方]. In conclusion, I firmly believe that [重申立场].
Only in this way can we [展望未来].
```

### 🔤 可替换句型

| 功能 | 句型 |
|------|------|
| 开头 | Recently, ... has become a hot topic. |
| 开头 | There is a heated debate over whether... |
| 观点A | Some people are in favor of the idea that... |
| 观点B | Others, however, take a negative attitude. |
| 我的观点 | As far as I am concerned, ... |
| 理由1 | First and foremost, ... |
| 理由2 | What's more / Moreover / In addition, ... |
| 举例 | A good case in point is that... |
| 让步 | It is true that... but... |
| 结论 | To sum up / In a word / All in all, ... |

## 📐 模板二：现象分析型（高频次之）

**适用题目：** 描述一个社会/校园现象 + 分析原因 + 你的看法

```
[第一段：描述现象]
In recent years, [现象] has become increasingly prevalent.
According to a recent survey, [数据/事实].
This phenomenon has drawn public attention.

[第二段：分析原因（2-3个）]
Several factors contribute to this trend.
First of all, with the rapid development of [相关领域], [原因1].
Moreover, [原因2] also plays a significant role.
Last but not least, [原因3].

[第三段：总结 + 建议]
From my perspective, [现象] is a double-edged sword.
On the one hand, [积极面]. On the other hand, [消极面].
Therefore, it is high time that we [建议/措施].
The government/schools/individuals should [具体行动].
```

## 📐 模板三：应用文/书信（偶尔考）

**适用题目：** 写一封信/邮件/通知/倡议书

```
[格式]
Dear [称呼],

[第一段：写信目的]
I am writing this letter to [目的：express my gratitude / invite you / 
make a complaint / apply for...].

[第二段：具体内容]
[原因/背景/详情阐述，3-4句话]

[第三段：总结+期待回复]
Thank you for your time and consideration.
I am looking forward to your early reply.

Yours sincerely,
[署名：Li Hua]
```

## 🎯 万能金句（随时救急）

### 开头万能句
```
1. With the rapid development of society, ... 在社会快速发展的背景下
2. Nowadays, there is a growing awareness that... 如今人们越来越意识到
3. When it comes to..., people's opinions differ. 当谈及...，人们观点各异
```

### 中间万能句
```
4. It goes without saying that... 不言而喻
5. There is no doubt that... 毫无疑问
6. As the saying goes, "Every coin has two sides." 俗话说，事物都有两面性
```

### 结尾万能句
```
7. Only by doing so can we... 只有这样我们才能...
8. It is high time that we took effective measures. 是时候采取有效措施了
9. All in all, we should strike a balance between... 总之要在...之间找到平衡
```

## ⚠️ 提分细节

| 细节 | 说明 |
|------|------|
| ✍️ 字迹 | 工整清晰！阅卷老师30秒扫一篇 |
| 📏 字数 | 120-180词，别少于120 |
| 📐 结构 | 三段式，段首空2格或顶格+段间空行 |
| 🔤 连接词 | First/Besides/Therefore 必须有 |
| ❌ 避免 | 全篇I think / very（换成extremely/significantly） |
| ✅ 加分 | 倒装句、强调句、定语从句各来一个 |

## 🔄 刘晓燕极简版模板（更简单更好背！）

> 来源：刘晓燕四级保命班 | 特点：词汇量要求极低，及格万能用

```
[第一段：2句话]
Nowadays, [话题] has become a hot topic.
Different people have different opinions on this issue.

[第二段：3-4句话]
Some people think that [观点A]. They believe that [理由].
However, others hold the opposite view. They argue that [理由].
As for me, I agree with the former / latter.

[第三段：2句话]
In conclusion, [你的观点].
Only in this way can we [展望].
```

## 🔤 王江涛词汇升级表（用这些词直接加分！）

> 来源：王江涛四级写作 | 普通词→升级词

| 普通词 | → | 升级词 |
|--------|---|--------|
| but | → | however / nevertheless / on the contrary |
| so | → | therefore / consequently / as a result |
| and | → | moreover / furthermore / additionally |
| I think | → | from my perspective / as far as I'm concerned |
| many | → | a great number of / numerous |
| important | → | significant / essential / crucial |
| good | → | beneficial / favorable / positive |
| bad | → | harmful / detrimental / negative |
| very | → | extremely / highly / remarkably |
| can | → | be able to / be capable of |
| should | → | be supposed to / it is advisable to |

> 💡 用法：写作时先在脑子里用普通词写完，再把标出来的地方替换成升级词

## ⚠️ 逻辑关系词位置规则

| 类型 | 位置 | 带逗号? |
|------|------|:--:|
| **However / Therefore / Moreover / Furthermore / In conclusion / For example / To sum up / Nevertheless / In my opinion / From my perspective** | **只放句首** | ✅ 必须跟逗号 |
| **because / since / so that / in order to / although / unless / while / whereas / as / when / if** | **只放句中** | ❌ 不加 |
| **but / and / so / or** | 句首或句中均可 | 句中不加 |

> ⚡ 口诀：不确定就往句首放，后跟逗号，永远正确！
