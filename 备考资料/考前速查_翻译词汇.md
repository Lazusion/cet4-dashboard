## 🆘 应急技巧

- 不会的词 → 上位词代替（包子→food）
- 不会的词 → 解释表达（针灸→treatment using needles）
- 不会的词 → 拼音也行（饺子→jiaozi）
- 长句子 → 切成 2-3 个短句
- 时态不确定 → 全用一般现在时
