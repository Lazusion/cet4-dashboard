## 🆘 翻译应急技巧

| 情况 | 应对 |
|------|------|
| 不会的词 | 上位词代替（包子→food） |
| 不会的词 | 解释性表达（针灸→treatment using needles） |
| 不会的词 | 用拼音！（饺子→jiaozi 也行） |
| 长句 | 切成2-3个短句，每个短句一个意思 |
| 时态不确定 | 全用一般现在时 |
